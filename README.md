# Universal Express React Redux Boilerplate

Personal boilerplate to start building apps based on ExpressJS / ReactJS and Redux

## Usage

### Development
```
$ npm run dev
```
### Development (SSR mode only)
```
$ npm run dev:ssr-only
```
### Build
```
$ npm run build
```
### Launch
```
$ npm start
```

## Dependencies

- Express 4
- React
- React router
- Redux
- React helmet
- Immutable
- SCSS
- Custom SCSS grid (mobile first)
- Icon font generator
- Flow
- Mocha / Chai
- Enzyme
